# National AI Validation Simulation

This repository contains the reproducible code and generated outputs for the national AI validation simulation used in the Predictive Representativity dissertation chapter.

## Main workflow

The canonical implementation is the R workflow.

```bat
"C:\Users\morales-fo.j\AppData\Local\Programs\R\R-4.5.3\bin\Rscript.exe" R\run_all.R "%C:\Users\morales-fo.j\Nextcloud\my-dissertation\national-AI-validation%"
```

The full workflow runs:

1. the finite-population simulation,
2. the Monte Carlo analysis,
3. validation checks,
4. publication figures,
5. LaTeX tables.

## Post-processing only

To regenerate manuscript tables and figures from existing R outputs:

```bat
Rscript -e "source('R/make_tables.R'); make_tables_R(normalizePath(getwd(), winslash='/'))"
Rscript -e "source('R/dgp.R'); source('R/make_figures.R'); make_figures_R(normalizePath(getwd(), winslash='/'))"
Rscript R\tr_etc_fixed_source.R "%C:\Users\morales-fo.j\Nextcloud\my-dissertation\national-AI-validation%" --overwrite
```

## Notes

The final manuscript tables and figures are generated from the R implementation. Python files are retained for comparison and historical cross-checking, but the dissertation outputs are based on the R workflow.

Raw replication files under `results/raw/` are not tracked because they are large and can be regenerated by the R workflow.
