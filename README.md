# National AI Validation Simulation

This repository contains the reproducible code and generated outputs for the
simulation study reported in the dissertation chapter:

> **Objective Reference Points, Predictive Representativity, and External Transportability**

This chapter is part of the dissertation:

> **Toward Safe AI**

Author: **Andrés Morales-Forero**

The study evaluates how different target-audit designs and estimators support
claims about target performance and external transportability of a locked
predictive system. The repository is organized to make the simulation workflow,
analysis decisions, generated tables, and manuscript figures reproducible.
